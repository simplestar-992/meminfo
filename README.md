## Meminfo

A useful command-line tool for developers.

### Installation

```bash
git clone https://github.com/simplestar-992/meminfo.git
cd meminfo
go build -o meminfo .
```

### Usage

```bash
./meminfo --help
```

---

## Quick Start

```bash
./meminfo [command]
```
