## ✨ Features

- 🚀 Blazing fast performance
- 💎 Clean and intuitive API
- 🔧 Zero configuration needed
- 🛡️ Secure by default
- 📦 Single binary, no dependencies

## 📥 Installation

```bash
git clone https://github.com/simplestar-992/meminfo.git
cd meminfo
go build -o meminfo .
./meminfo --help
```

## 🚀 Quick Start

```bash
./meminfo -h
```

## 🧪 Examples

```bash
# Example 1
./meminfo status

# Example 2  
./meminfo --verbose

# Example 3
./meminfo help
```
